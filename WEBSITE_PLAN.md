# Go7Studio.com Website Plan

## Mission
Create a stunning, SEO-optimized website for Go7Studio that showcases games and apps, drives traffic, and converts visitors into players/users.

## Tech Stack
- **Framework:** Next.js 14 (App Router) - best for SEO, Vercel-native
- **Styling:** TailwindCSS + custom animations
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel (go7studio.com)

## Design Language (from existing Go7Landing)
- Dark theme with glass-card aesthetic
- Purple (#8B5CF6) / Blue (#3B82F6) accent gradients
- Animated gradient backgrounds
- Smooth scroll animations
- Modern, playful but professional

## Pages Structure

### 1. Home (/)
- **Hero Section:** Bold headline, animated background, CTA buttons
- **Featured Games:** Showcase Empire Tycoon, SlimeSlip, Pet Paradise
- **About Section:** Brief Go7Studio story
- **Stats/Social Proof:** Downloads, ratings, player counts
- **Newsletter/Contact:** Email capture

### 2. Games (/games)
- Grid of all games with filters
- Each card: thumbnail, name, platform badges, rating, CTA

### 3. Individual Game Pages
- **/games/empire-tycoon** - Android tycoon game
- **/games/slimeslip** - Roblox obstacle game (Coming Soon)
- **/games/pet-paradise** - Roblox pet breeding (Coming Soon)

Each game page includes:
- Hero with gameplay trailer/screenshots
- Features list
- Download/play buttons
- Reviews/ratings
- Related games

### 4. About (/about)
- Go7Studio story
- Team (if applicable)
- Mission/values

### 5. Contact (/contact)
- Contact form
- Social links
- Business inquiries

### 6. Legal
- /privacy-policy
- /terms-of-service
- /games/[game]/privacy-policy (per-game policies)
